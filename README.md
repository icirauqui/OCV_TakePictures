# OCV_TakePictures

Take pictures for further use.

HowTo:

 1 - Clone and compile:

    git clone https://github.com/icirauqui/OCV_TakePictures.git OCV_TakePictures
    
    cd OCV_TakePictures
    mkdir build
    cd build
    cmake ..
    make

 2 - Usage:
 
   Run the program: 
   
    .takepictures
 
   Follow the instructions in the terminal.
